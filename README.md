# Anycubic-Kobra-3-Max
This project is realted to Anycubic Kobra 3 Max ***Layer shifting*** issue after ***color change***!

## How to use this repo

1. Copy code from `Layer-shift-gcode.md`.
2. Open AnycubicSlicerNext.
3. Open Printer settings.
4. Open Machine G-code.
5. Scroll down to `Change filament G-code`
6. Replace the G-code there with our solution.
7. Save the changes to a new profile and move to testing.

## Testing
This solution should fix the issue (Multi-color printing).
